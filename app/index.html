This microservice will return formatted data taken from rcpt-saver for visualization

  • go to /api/piechart to see a categorization of bills
  • get data time-based data at /api/timechart/option?params

    option

    • /weekly

      • params: startDate=dd-mm-yyyy

      i.e. /weekly?startDate=01-20-2019

    • /monthly

      • params: startDate=dd-mm-yyyy

      i.e. /monthly?startDate=01-20-2019

    • /yearly

      • params: year=yyyy

      i.e. /yearly?year=2019
